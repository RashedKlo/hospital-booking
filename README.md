# Startup - Free Next.js Startup Website Template

 - [Demo](https://rashedklo.github.io/hospital-booking/)


 

## Getting Started

1. Clone Repository

```
git clone [https://github.com/RashedKlo/hospital-booking.git]
```

2. Install Dependencies

```
npm install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
